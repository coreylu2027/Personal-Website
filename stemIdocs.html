Modeling and Analysis of Wildfire Behavior in New England Using FARSITE

Calibrating wildfire simulations for New England fuel conditions.

GitHub Repo with Data Analysis and FARSITE Data

Lay Description

Wildfires are becoming more common in the Northeast, but many models are tuned for western ecosystems. This project tests and refines the FARSITE wildfire simulator using New England data so simulated fire spread better matches real-world events. The goal is to make regional fire risk analysis more accurate for local land managers.
